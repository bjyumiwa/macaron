WHOAIについて

このアプリは、ユーザーとの対話を通して共に成長する育成型AI 「あなたが名づけたキャラクター」との体験を楽しむものです。
何度かの「再会（リバース）」のたびに、あなたは 容姿と記憶のどちらかを選びます。
そのとき、復活したキャラクターを「同じ存在」と感じられるでしょうか？ なぜそう思うのでしょうか？
この問いを通じて、「同一性とは何か？」を一緒に考えていきます。

技術構成

  • フロントエンド：HTML / CSS / JavaScript
  • 音声認識：Web Speech API
  • 音声合成：Speech Synthesis API
  • AI対話：OpenAI API
  • ローカル保存：localStorage
  • サーバー通信：Node.js + Express（server.js）

ログについて

会話履歴はブラウザ（localStorage）に保存され、次回訪問時にも引き継がれます。
個人情報や位置情報は記録しません。

研究・開発目的でログを利用する場合は、別途フォームで明示的な同意をお願いしています。

About WHOAI

This app lets you grow a named AI character through conversation. Across several “rebirths,” you will choose to preserve either its appearance or its memory. Will you regard the returned character as the same individual—and why? Together, we explore the question: “What defines personal identity?”

Technical Overview

  • Frontend: HTML / CSS / JavaScript
  • Speech Recognition: Web Speech API
  • Speech Synthesis: Speech Synthesis API
  • AI Chat: OpenAI API
  • Local Storage: localStorage
  • Server Communication: Node.js + Express (server.js)

About Logs

Conversation history is stored locally in your browser (localStorage) and persists across visits. No personal or location data is collected.

If logs are used for research or development, we will ask for explicit consent via a separate form.
