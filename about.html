WHOAIについて

本プロジェクト「WHOAI」は、人工知能と人間の関係性について考えるための研究型アプリケーションです。 利用者はキャラクターと対話を重ね、その成長や「再生」を通じて、自己同一性（アイデンティティ）の在り方を探ります。

キャラクターが再生する際、容姿を残すか／記憶を残すかを選ぶことになります。 その選択は「同じ存在」と言えるのか？という根本的な問いを投げかけます。

研究目的

本研究の目的は、AIとの関わりを通じて人間がどのように「個人の同一性」を認識するのかを探ることです。 特に、容姿や記憶の変化が「存在の連続性」にどのような影響を与えるかを考察します。

技術構成

  • フロントエンド：HTML / CSS / JavaScript
  • AI対話：OpenAI API
  • 音声入出力：Web Speech API, Speech Synthesis API
  • ローカル保存：localStorage
  • サーバー通信：Node.js + Express

倫理的配慮とログについて

会話履歴は利用者のブラウザ（localStorage）にのみ保存されます。 個人情報や位置情報は収集していません。

研究目的でログを利用する場合は、必ず事前に明示的な同意をお願いしています。 研究は倫理的配慮に基づき実施されます。

About WHOAI

The WHOAI project is a research-oriented application designed to reflect on the relationship between humans and artificial intelligence. By conversing with an evolving AI character, users explore the question of personal identity.

Upon each rebirth, you choose whether to preserve the character’s appearance or its memory. This raises the fundamental question: “Is it still the same being?”

Research Goal

The aim is to study how humans perceive continuity of identity in relation to AI, especially how changes in memory or appearance affect that perception.

Technical Overview

  • Frontend: HTML / CSS / JavaScript
  • AI Chat: OpenAI API
  • Speech Input/Output: Web Speech API, Speech Synthesis API
  • Local Storage: localStorage
  • Server: Node.js + Express

Ethics and Logs

Conversation history is stored only in the browser (localStorage). No personal or location data is collected.

Any use of data for research requires explicit consent in advance. The study is conducted under ethical guidelines.
