WHOAIについて

このアプリは、ユーザーとの対話を通して共に成長し、最終的には別れの選択をする育成型AI「あなたが付けた名前のキャラクター」との体験を楽しむためのものです。
何度も生まれ変わり復活するとき、「容姿」か「記憶」かのどちらかを選ぶことができます。
あなたは、その復活したキャラクターを同一人物だと思いますか？ それは、どんな理由からそう思いますか？
この問いを通じて、「同一性とは何か？」を考える研究にご協力いただける方は、どうぞ次にお進みください。

技術構成

  • HTML / CSS / JavaScript
  • 音声認識：Web Speech API
  • 音声合成：Speech Synthesis API
  • AI対話：OpenAI API（gpt-3.5 または gpt-4）
  • ローカル保存：localStorage
  • サーバー通信：Node.js + Express（server.js）

ログについて

会話履歴は、ブラウザ内（localStorage）に保存され、次回訪問時にも引き継がれます。
個人情報や位置情報は一切記録されませんのでご安心ください。

開発・研究目的でログを使用する場合には、別途フォームにて同意をいただいた上で実施します。

About WHOAI

This app offers an experience where you interact with a growing AI character—one you name yourself. As the character is reborn multiple times, you are asked to choose between preserving its appearance or its memory. Do you still consider the reborn character to be the same individual? What makes you think so? If you are willing to reflect on the question, “What defines personal identity?”, please proceed.

Technical Overview

  • HTML / CSS / JavaScript
  • Speech Recognition: Web Speech API
  • Speech Synthesis: Speech Synthesis API
  • AI Chat: OpenAI API (gpt-3.5 or gpt-4)
  • Local Storage: localStorage
  • Server Communication: Node.js + Express (server.js)

About Logs

Your conversation history is stored locally in your browser (localStorage) and will persist across visits. No personal or location data is ever collected, so you can use the app with peace of mind.

If your logs are to be used for research or development purposes, we will request your explicit consent through a separate form.
